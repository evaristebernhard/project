# 实验7 — 矩阵乘法 (test7)

说明
- `test7.f90` 实现了矩阵乘法（A × B = C）。程序通过 `cmd.par` 读取三个文件名（A、B、C），然后读取矩阵大小和数据，完成乘法并输出结果到文件和屏幕。

程序流程与文件格式
1. `cmd.par` 中应有三行，每行形如 `标签 文件名`，例如：
   ```
   A_file A.dat
   B_file B.dat
   C_file C.out
   ```
   程序使用 `read(20,*) str, A_file` 的方式读取。
2. 矩阵数据文件（如 `A.dat`）第一行是 `m,n`（行列数），接着是矩阵数据，每行 `n` 个数。

关键子程序
- `read_mn`：读取矩阵尺寸 m,n。
- `read_data`：读取矩阵数据到二维数组（注意目前实现中在读矩阵前有一句 `read(40,*) r1,r2` 用来跳过头两项 — 若你的数据从第一行就开始，则可能需要删除这行）。
- `mul`：三重循环实现矩阵乘法（注意循环索引 `i,j,r` 都应为整数）。
- `output`：把结果写入文件并在屏幕打印。

如何编译与运行
```bash
cd fortran实验/实验7
gfortran test7.f90 -o test7
./test7
```

课堂提示
- 矩阵乘法需要 `A_n == B_m` 才能执行；程序会检查这一点并在不匹配时提示错误。
- 对初学者建议：先用小矩阵（例如 2×3 与 3×2）手工验证输出是否正确。
