# 实验3 — 指数展开 (exp)

说明
- `exp.f90` 使用泰勒级数展开计算 e^x，通过累加各项直到一项小于阈值或达到最大迭代次数。

代码要点（初学者应关注）
- 循环结构：`DO WHILE (condition)` 用于迭代直到满足条件。
- 逐项计算：使用 `term = term * x / n` 来构造下一项（节省阶乘计算）。
- 内置函数：`EXP(x)` 可用于对比标准库结果。

如何编译与运行
```bash
cd fortran实验/实验3
gfortran exp.f90 -o exp
./exp
```

练习建议
- 修改 `x` 和 `nmax` 的值，观察级数收敛速度（正/负/大 x 的差别）。
