# 实验6 — 类似的裁剪程序（test1.f90）

说明
- `test1.f90`（实验6）与实验5/4 的裁剪程序类似：通过 `cmd.par` 指定文件与范围，统计记录、读入内存、筛选并写出结果。

要点总结
- 代码使用 `OPEN`/`READ`/`CLOSE` 进行文件操作；用动态数组 `ALLOCATE` 存放中间数据。
- 注意循环索引应显式声明为整数（例如 `integer x,y,i`），否则编译器可能发出警告或采用隐式类型规则导致错误。

如何编译与运行
```bash
cd fortran实验/实验6
gfortran test1.f90 -o test1
./test1
```

课堂风格建议
- 保持 `cmd.par` 的格式与程序读取方式一致（程序里读法一般是 `read(10,*) str, filename`，即先读一个标签再读文件名）。
